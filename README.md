# Wyvern Installer
Package for installation of [Wyvern theme](https://github.com/madesane/wyvern)

Install globally
```
npm install -g wyvern-installer
```

Than use command
```
wyvern
```

With wyvern-installer you could install:
* Wordpress + Wyvern + Wyvern Child
* Wordpress + Wyvern + Wyvern Shop
* Wordpress + Wyvern
* Wyvern + Wyvern Child
* Wyvern
